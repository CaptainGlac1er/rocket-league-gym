make([game_speed, tick_skip, spawn_opponents, ...])

param game_speed

The speed the physics will run at, leave it at 100 unless your game can't run at over 240fps
rlgym.make.make(game_speed=100, tick_skip=8, spawn_opponents=False, self_play=None, team_size=1, gravity=1, boost_consumption=1, terminal_conditions=(common_conditions.TimeoutCondition(225), common_conditions.GoalScoredCondition()), reward_fn=DefaultReward(), obs_builder=DefaultObs(), action_parser=DefaultAction(), state_setter=DefaultState(), launch_preference=LaunchPreference.EPIC, use_injector=False, force_paging=False, raise_on_crash=False, auto_minimize=False)
Parameters

  • game_speed (int) – The speed the physics will run at, leave it at 100 unless your game can’t run at over 240fps

  • tick_skip (int) – The amount of physics ticks your action will be repeated for

  • spawn_opponents (bool) – Whether you want opponents or not

  • team_size (int) – Cars per team

  • gravity (float) – Game gravity, 1 is normal gravity

  • boost_consumption (float) – Car boost consumption rate, 1 is normal consumption

  • terminal_conditions (List[object]) – List of terminal condition objects (rlgym.utils.TerminalCondition)

  • reward_fn (object) – Reward function object (rlgym.utils.RewardFunction)

  • obs_builder (object) – Observation builder object (rlgym.utils.ObsBuilder)

  • action_parser (object) – Action parser object (rlgym.utils.ActionParser)

  • state_setter (object) – State Setter object (rlgym.utils.StateSetter)

  • launch_preference (str) – Rocket League launch preference (rlgym.gamelaunch.LaunchPreference) or path to RocketLeague executable

  • use_injector (bool) – Whether to use RLGym’s bakkesmod injector or not. Enable if launching multiple instances

  • force_paging (bool) – Enable forced paging of each spawned rocket league instance to reduce memory utilization immediately, instead of allowing the OS to slowly page untouched allocations. WARNING: This will require you to potentially expand your Windows Page File [1] and it may substantially increase disk activity, leading to decreased disk lifetime. Use at your own peril. Default is off: OS dictates the behavior.

  • raise_on_crash (bool) – If enabled, raises an exception when Rocket League crashes instead of attempting to recover. You can attempt a recovery manually by calling attempt_recovery()

  • auto_minimize (bool) – Automatically minimize the game window when launching Rocket League

Returns

Gym object

[1]: https://www.tomshardware.com/news/how-to-manage-virtual-memory-pagefile-windows-10,36929.html
