is_terminal(current_state: rlgym.utils.gamestates.GameState) → bool

Check to see if the game score for either team has been changed. If either score has changed, update the current known scores for both teams and return True. Note that the known game scores are never reset for this object because the game score is not set to 0 for both teams at the beginning of an episode.
